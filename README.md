# Snake

Snake is a classic game coded in Python. It runs on your console screen. Yes! ASCII

## Getting Started

These instructions will get you a copy of the 'project' up and running on your local machine.

### Prerequisites

Python. That's all!

### Installing

Just clone it...

## Running

python snake.py

### Moving the snake

Use the arrows.

### Quit the game

The letter 'q' quit the game.
